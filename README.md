# Nighta

language: [中文](./README_CN.md) | [ENGLISH](.)

`Nighta` is a lightweight scripting language, with its name signifying "something created overnight."

## Code Sample

```nighta
  (var msg1 "Hello,")
  (var msg2 "World!")
  (say (+ msg1 msg2))
```
