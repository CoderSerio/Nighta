`Nighta` 是一款轻量级的脚本语言。其名字的寓意是“一夜之间产生的事物”。

## 代码示例

```nighta
  (var msg1 "Hello,")
  (var msg2 "World!")
  (say (msg1 + msg2))
```

## 快速开始
